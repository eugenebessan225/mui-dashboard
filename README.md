## Built With

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm

### Installation

1. Clone the repo

```sh
git clone https://github.com/td2thinh/mui-dashboard.git
```

2. Install dependencies

```sh
npm install
```

## Usage
Run app
```sh
npm run dev 
```
App available at http://localhost:PORT
